h1. Install Nginx

h3. Simple standalone instance:

Create a nginx instance listen on port 8080:

<pre>
  nginx {"myapp":}
</pre>

h3. Possible paramiters:

<pre>
- upstream_tipe [proxy,fastcgi,none]
- upstream_url  [host:port]
</pre>

Example to create a proxy upstream:

<pre>
  nginx {"myapp":
    upstream_tipe  => "proxy",
    upstream_url   => "http://localhost:8090",
  }
</pre>
